# TerminalLauncher

A minimal Android launcher. Home screen is empty (just your wallpaper + system UI).
Swipe up anywhere to open the command palette.

## Commands
- `list apps` — lists all installed apps
- `launch [name]` — opens an app (partial names work, e.g. "launch cam")
- `clear` — clears the terminal output
- `help` — shows available commands
- Back button — returns to home
  
## Developer note
Even though i had no intention to make this app compatible with other devices, the pure minimalism of this app led it to become widely compatible, somehow. I suppose it can handle Android 8+.
